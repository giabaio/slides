Some maths

We can also include maths, using standard \(\rm\LaTeX\):

\[\class{italian-green}{p(y\mid\mathcal{D}) = \int p(y\mid\bm\theta)p(\boldsymbol\theta\mid\mathcal{D})d\boldsymbol\theta} \]

To use colours with \(\rm\LaTeX\), needs a little hack: $\class{color}{expression}$

You can define a probabilistic assumption as \[y \mid \bm\theta \sim \dpois(\bm\theta)\] but in line it would look like this \(\color{MidnightBlue}{y\sim \dnorm(\mu,\sigma^2)}\)

These use the predefined \(\rm\LaTeX\) macros, saved in the file latex_macros.html. More can be (re)defined by simply adding to the file, which gets pre-loaded at the beginning of the resulting .html output; example: $\DIC=2345$ is typeset as \(\DIC=2345\).

For instance, the command $\dnorm$ translates to the text Normal (typeset in the current font). But can also use $\txt{Some text}$, which gets rendered as \(\txt{Some text}\) ‚Äî it‚Äôs like \text{...} but with the current font
